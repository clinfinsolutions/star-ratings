# 2026 Hospital Star Ratings Explorer

A free tool from [Clinfin Solutions](https://www.linkedin.com/in/deanna-payne) for healthcare quality leaders.

Search any hospital. See if their HAC penalty status moved year over year. See where 2026 Star Ratings will land when CMS releases them on May 13.

## Live URL

https://clinfinsolutions.github.io/star-ratings/

## Data Sources

- **Hospital General Information** — CMS Care Compare, 2025 Release
- **HAC Reduction Program** — CMS, FY2025 + FY2026

Both files publicly available at [data.cms.gov](https://data.cms.gov/provider-data/topics/hospitals).
